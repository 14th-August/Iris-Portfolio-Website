# Iris-Portfolio-Website
A Freelance SEO website for a makeup artist who I know. The project will also support backend client capabilities and finding clients through postings.

# File Structure

## /next-project | Name of the project.
   - > .next | Built on npm run dev
   - > app | root level directory of the project, contains routing and pages
   - > components | contains ui elements
   - > node_modules | all of the installed dependencies
   - > public | contains all of my photos
   - > utils | contains ts arrays of photos to use
   - .gitignore | ignore commits
   - AGENTS.md | agents log file
   - CLAUDE.md | claude log file
   - eslint.config.mjs | linter file
   - next-env.d.ts | next dev file
   - next.config | any outside source imports eg: https example.com
   - package-lock.json | dependency version in local
   - package.json | same thing just range
   - postcss.config.mjs | tailwind import config
   - README.md | this
   - tsconfig.json | compiler/linter
